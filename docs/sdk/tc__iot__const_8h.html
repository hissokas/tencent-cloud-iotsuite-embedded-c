◆ tc_iot_sys_code_e

typedef enum _tc_iot_sys_code_e tc_iot_sys_code_e

系统常见错误码定义，基本定义原则：

  1. =0 表示SUCCESS状态，即函数调用执行成功
  2. <0 表示系统出现异常，返回值为对应如下错误码
  3. >0 用于读写、收发类函数，表示当前操作对应执行成功的字节数。
