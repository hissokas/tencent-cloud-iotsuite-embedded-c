enum _tc_iot_log_level_e

日志等级 TRACE < DEBUG < INFO < WARN < ERROR < FATAL < OFF

枚举值
TC_IOT_LOG_TRACE 

输出各类详细过程及输入输出信息

TC_IOT_LOG_DEBUG 

输出应用调试信息

TC_IOT_LOG_INFO 

输出从粗粒度上，描述了应用运行过程

TC_IOT_LOG_WARN 

输出潜在的有害状况，需要及时关注

TC_IOT_LOG_ERROR 

输出错误事件，但应用可能还能继续运行

TC_IOT_LOG_FATAL 

输出非常严重的错误事件，可能会导致应用终止执行

TC_IOT_LOG_OFF 

关闭所有日志
